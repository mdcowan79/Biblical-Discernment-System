# 🛡️✝️ Biblical Discernment System (BDS)

A simple, user-friendly tool to help believers evaluate teachings and ideas against Scripture, Christ, and the Gospel.

## Features

- **Search Engine Interface** - Simple, clean design for describing teachings
- **Multi-Input Support** - Text descriptions, links, or video information
- **16-Category Theological Analysis** - BDT (Deep Test)
- **8 Clear Verdicts** - BDX (Quick Check)
- **Instant Results** - Category-by-category breakdown
- **Mobile-Friendly** - Works on all devices
- **No Sign-Up Required** - Free, private, accessible

## The 16 Categories

1. Theology Proper
2. Christology
3. Pneumatology
4. Trinity
5. Bibliology
6. Anthropology
7. Hamartiology
8. Soteriology
9. Ecclesiology
10. Eschatology
11. Angelology
12. Covenant & Redemptive History
13. Christian Identity
14. Christian Ethics
15. Christian Practice
16. Cultural & Philosophical Claims

## The 8 Verdicts

- 🟩 **Biblical** - Fully consistent with Scripture
- 🟩 **Biblical but Incomplete** - True but missing context
- 🟨 **Mixed** - Contains both truth and error
- 🟨 **Distorted** - Biblical language, altered meaning
- 🟨 **Speculative** - Beyond Scripture's revelation
- 🔴 **Unbiblical** - Directly contradicts Scripture
- 🔴 **Deceptive** - Appears biblical but misleads
- 🔴 **Anti-Gospel** - Attacks or replaces the gospel

## How to Use

1. Open the app
2. Describe a teaching, paste a link, or describe a video
3. Answer 16 simple yes/no/partial questions
4. Get an instant verdict with detailed analysis
5. Review category-by-category breakdown

## Installation & Deployment

This is a static website with no server requirements. Deploy to:

- **GitHub Pages** (free)
- **Netlify** (free)
- **Vercel** (free)
- Any web hosting

### GitHub Pages Deployment

1. Go to Settings → Pages
2. Set Source to `main` branch
3. Save
4. Site will be live at `https://mdcowan79.github.io/Biblical-Discernment-System/`

## Customization

Easy to modify:

- **Change questions** - Edit BDS_CONFIG in `app.js`
- **Adjust verdicts** - Modify verdict colors, explanations in `app.js`
- **Update styling** - Edit `styles.css`
- **Add content** - Edit sections in `index.html`
